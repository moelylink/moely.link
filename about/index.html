关于网站

关于“萌哩”

moely由moe和ly组成，名字灵感来自于bitly。中文音译为萌哩，灵感来自劈里啪啦。图标灵感来自某P站。

为了保护画师的相关权益，本网站一切内容禁止用于AI训练！

网站源代码已经在 GitHub 上开源，喜欢的话star支持一下！

本站域名： moely.link

我们的Telegram频道： @gongzhutonghao

网站特性

  • 响应式瀑布流设计
  • 图片懒加载、图片灯箱效果
  • 详情页图片信息（画师、描述等）展示
  • 通过标签筛选图片
  • 每张图均配有来源链接
  • 支持最高4K分辨率的原图下载
  • PWA（渐进式网络应用程序）支持
  • 随机传送功能，每次一点小惊喜～

关于更新

本网站内容均精选自 Pixiv 和 Twitter（现称X） ，经过人工筛查后推送，或许更新频率不是很高，但是我们注重每一张图的质量。

默认情况下，本站图片按照作者发布日期（由新到旧）排序。
