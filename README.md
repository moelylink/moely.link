# 萌哩 moely.link

萌哩 - 萌萌的二次元美图。全网精选二次元美图，提供高质量原图下载。收藏美图，收获美好。

## 网站特性

- 响应式瀑布流设计
- 图片懒加载、图片灯箱效果
- 详情页图片信息（包括画师、描述等）展示
- 配有原图链接（指向Pixiv）和下载按钮（可直接下载原图，分辨率最高可达4K）
- PWA（渐进式网络应用程序）支持

## 技术栈

- HTML
- CSS
- JavaScript
